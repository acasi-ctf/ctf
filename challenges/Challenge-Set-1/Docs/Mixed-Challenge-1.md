# Mix_Challenge_1
This is the mix challenge for classical cipher, so you will use your knowledge about classical cipher.

## Recall
### Letter to Number
Example: A = 1, B = 2, C = 3,and so on.

### Morse Code
Example: A = ". -", B = "- . . .", C = "- . - .", and others.

### Reverse
Example: ABC = CBA

### Caesar
Example: if n = 5, "HELLO" = "MJQQT"
